# 💫 Sobre Mí

¡Hola! 👋 Soy **José Daniel Loza Marín**, desarrollador Full Stack y especialista en ciberseguridad apasionado por crear aplicaciones web seguras y de alto rendimiento.

📅 **Edad:** 21 años (nacido el 15 de noviembre de 2004)  
🎓 **Educación:** Estudiante en la Universidad Tecnológica de Xicotepec de Juárez  
🌍 **Ubicación:** México

### 🎯 ¿Quién Soy?

Soy un desarrollador Full Stack con una pasión ardiente por la **ciberseguridad** y el **desarrollo web moderno**. Me especializo en:

- 🔐 **Ciberseguridad:** Pruebas de penetración, análisis de vulnerabilidades, hardening de aplicaciones
- 💻 **Frontend:** Desarrollo de interfaces modernas y responsivas con React, TypeScript y TailwindCSS
- 🛠️ **Backend:** APIs robustas con Express.js, Node.js y bases de datos relacionales
- 🎨 **Diseño:** Experiencias visuales atractivas que combinan funcionalidad y estética

### 💡 Mi Filosofía

Creo que la mejor tecnología es aquella que es **segura, accesible y hermosa**. Cada línea de código que escribo tiene como objetivo hacer el ciberespacio más seguro mientras creo experiencias web excepcionales.

### 📚 Siempre Aprendiendo

Me encanta experimentar con nuevas tecnologías, resolver problemas complejos y enfrentar desafíos que me hagan crecer como desarrollador. En mi tiempo libre, profundizo en temas de seguridad ofensiva y defensiva.
